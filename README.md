# 계산기 과제
<br>

## 📑 소개
자바 문법 실습 및 객체 지향 개념 활용 과제
<br><br>

## 📆 기간
* 25/07/10 ~ 25/07/14
<br>

## ⚙ 개발 환경
* JDK 17.0.12
<br>

## 🔧 주요 기능
[1단계] : 클래스 없이 자바의 기본 문법만을 사용하여 구현한 계산기

[2단계] : 객체 지향 개념을 적용하여 클래스 정의, 연산을 수행하는 메소드 작성한 계산기

[3단계] : 자바의 Enum, 제네릭, 람다 & 스트림을 적용한 계산기

[추가 기능] : 필터링에 비교연산자를 받는 기능 추가
